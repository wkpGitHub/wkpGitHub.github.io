# 扫雷游戏

一个使用原生 HTML、CSS 和 JavaScript 实现的经典扫雷游戏。

## 功能特点

### 🎮 游戏功能
- **经典扫雷玩法**：点击扫雷，右键标记
- **难度选择**：简单(9×9,10雷)、中等(16×16,40雷)、困难(16×30,99雷)
- **自定义难度**：可以自定义行数、列数和地雷数量
- **自动扩展**：点击空白格子自动展开周围区域
- **智能标记**：显示剩余地雷数量

### ⚡ 交互体验
- **精美UI设计**：渐变背景、现代化界面
- **动画效果**：按钮悬停、点击反馈
- **响应式布局**：适配移动设备
- **游戏状态**：实时显示剩余地雷数和游戏时间

### 🎯 其他特性
- **计时器**：记录游戏时长
- **游戏结束提示**：胜利/失败弹窗
- **防止误操作**：第一次点击不会踩雷
- **右键菜单屏蔽**：防止游戏时弹出右键菜单

## 文件结构

```
minesweeper/
```

## 游戏开发说明

### 核心算法
- **布雷算法**：随机分布地雷，避开第一次点击位置和相邻格子
- **递归展开**：点击空白格子时自动展开相邻空格
- **获胜判断**：所有非地雷格子被揭示即获胜
- **失败判断**：点击到地雷即失败

### 技术实现
- 使用原生 JavaScript 类进行面向对象编程
- 事件监听实现交互功能
- setInterval 实现计时器
- CSS Grid 布局实现游戏面板
- Flexbox 实现响应式布局

## 使用方法

1. 在浏览器中直接打开 `index.html` 即可开始游戏
2. 选择难度或自定义游戏参数
3. 点击"新游戏"按钮开始
4. 左键点击扫雷，右键点击标记
